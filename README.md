# Vocabulary builder

A simple vocabulary trainer I made to help with my german learning.
As of now it is very barebones and I am using Python with streamlit, initially I had a plan to use Java but I realiased it would be easier to conjure Goethe himself than that...
Basing this on dict.cc and using the same format of their lists for seamless transitioning, also want to put Leitner system in the future but lets see what happens.


## Files
vb.py (streamlit vocabular trainer)
vocb.py (parser)
vocab.csv (list using dict.cc format)

## Dependencies
streamlit
*run with python -m streamlit run ..../vb.py*
